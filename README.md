# WebRadio-Guy-Chauvet

## **Description du projet**

Ceci est le github du projet WebRadio du lycée Guy Chauvet pour la web radio du lycée<br>

***
## **Environnement technique**
**Serveur utiliser** :
- Apache 2.4

**SGBD** :
- MariaDB 10.5.18
- Adminer 4.8.1

**Langages** :
- HTML 5
- CSS 3
- PHP 7
- JavaScript  

**Accès** :
- FTP 3.0
- Sauvegarde
- Planification a prévoir/recherche
