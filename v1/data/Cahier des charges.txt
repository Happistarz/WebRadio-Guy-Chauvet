LECTEUR AUDIO:
Les lecteurs audio doivent avoir un bouton “favoris”, un bouton“lecture” et un bouton “pause”. Il doit contenir le titre du podcast, le(s) nom(s) des chroniqueurs et le temps que durent les podcasts. Il devra aussi contenir la date de publication du podcast.

CONVENTION DE NOMMAGE:

TOUS LES COMMENTAIRES DOIVENT ÊTRE  ÉCRITS EN ANGLAIS 

Pour le CSS:
style.css → Le main du projet css
reset.css → Au cas où la page ne fonctionne pas

Pour le HTML:
index.html → Site main 
Dossier html → Contient toutes les pages
[NomDePage].html →  format de nommage des fichiers html

Pour la base de données:
Dossier data → fichier contenant toutes les bases de données
BDD-[NomDePage].php → répertoire des comptes

Pour le PHP:
Dossier php → Le site sur internet
[NomDePage].php → format de nommage des fichiers PHP

Pour Java Script
Dossier JS → fichier qui permettra l’animation du loader
[NomDePage].js → format de nommage des fichiers JS

Pour les images:
Dossier image → Contient toutes les images qui serviront pour le site
*.PNJ → format de nommage 
*.JPEG → format de nommage
